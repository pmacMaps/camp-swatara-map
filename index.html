Camp Swatara Hiking Trails Map

Below are links to two versions of an interactive webmap of hiking trails at Camp Swatara. One was built with Leaflet.js, and the other with the ArcGIS JavaScript API. View the project on GitHub.

There are plans to eventually create this simple map using Mapbox.js and Google Maps API. My goal is to have a repository that can show how the same web map is built using various JavaScript libraries.
