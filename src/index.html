Camp Swatara Hiking Trails

About the Map

This is a concept for a web map of the hiking trails at Camp Swatara.

I envision the web map being embedded on a page on Camp Swatara's website.

We could add the camp roads, points of interest, and Appalachian Trail, if desired.

The "Select Basemap" link opens a tool to change basemaps. I was thinking we could have a topographic and imagery basemap.

You can also click on a trail for information about blaze and distance.

There are couple options for embedding this on the Camp's website:

  1. Use an iframe whose source is this GitHub page, or files on the Camp Swatara website server.
  2. Place the files that create the map on the website according to the best practices of the website's platform.
